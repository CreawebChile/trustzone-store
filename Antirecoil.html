#        Guía de Uso del Script         #
#       Configuración Anti-Recoil       #
#########################################

Este script de AutoHotkey te permite ajustar la sensibilidad y la compensación de retroceso en tu sistema a través de una interfaz gráfica sencilla. A continuación, se explica cómo funciona el script y cómo utilizarlo.

=========================================
REQUISITOS PREVIOS
=========================================
- **AutoHotkey**: Debes tener instalado AutoHotkey en tu computadora para ejecutar este script.
 
  Descarga el archivo: AutoHotkey_1.1.37.02_setup
                         O
  Descárgalo desde: https://www.autohotkey.com/

=========================================
UBICACIÓN DEL ARCHIVO
=========================================
- **Archivo del Script**: Guarda el archivo del script en el Escritorio de tu computadora para facilitar su acceso.
  - Nombre del archivo: `AntiRecoil.ahk`
  - **Acceso Rápido**: Al tener el archivo en el Escritorio, podrás encontrarlo y abrirlo fácilmente cuando lo necesites.

- **Carpeta de Sonidos**: La carpeta que contiene los sonidos necesarios para el script debe estar ubicada en el disco local **C:**. Asegúrate de que la ruta sea correcta para evitar errores en la ejecución del script.

=========================================
DESCRIPCIÓN DEL SCRIPT
=========================================
- **Sensibilidad y Compensación de Retroceso**:
  Permite ajustar estos valores desde la interfaz gráfica.

- **Valores Predeterminados**:
  La sensibilidad predeterminada es 2 y la compensación de retroceso es 100.
  Estos valores se pueden ajustar, pero se restablecen al reiniciar el script o al presionar "Restaurar Valores".

- **Atajos de Teclado**:
  Puedes usar atajos de teclado para modificar la sensibilidad y la compensación sin necesidad de abrir la interfaz.

=========================================
INSTRUCCIONES DE USO
=========================================
1. **Iniciar el Script**
   - Haz doble clic en el archivo del script `AntiRecoil.ahk` en el Escritorio para ejecutarlo.
   - Aparecerá una ventana con la interfaz gráfica para hacer ajustes.

2. **Ajustar Sensibilidad y Compensación**
   - **Usar Sliders**: Ajusta la sensibilidad (1-10) y la compensación (0-200) con los sliders.
   - **Campos de Texto**: También puedes ingresar valores directamente en los campos de texto.
   - **Restaurar Valores**: Presiona el botón "Restaurar Valores" para volver a los valores predeterminados.

3. **Usar Atajos de Teclado**
   - **F9**: Aumentar Sensibilidad
   - **F10**: Disminuir Sensibilidad
   - **F11**: Aumentar Compensación de Retroceso
   - **F12**: Disminuir Compensación de Retroceso
   - **F7**: Activar/Desactivar Script Anti-Recoil
   - **F8**: Desactivar completamente el script
   - **F6**: Activar Modo Silencioso (sin sonidos)

4. **Guardar la Configuración**
   - La configuración se restablece al predeterminado cada vez que reinicias el script o presionas "Restaurar Valores".

5. **Detener el Script**
   - Haz clic derecho en el icono de AutoHotkey en la bandeja del sistema (junto al reloj) y selecciona "Exit" para cerrar el script.

=========================================
EJEMPLO DE USO
=========================================
1. **Inicia el Script**: Doble clic en `AntiRecoil.ahk` en el Escritorio.
2. **Ajusta Sensibilidad**: Usa el slider para cambiar el valor.
3. **Restaurar Valores**: Haz clic en "Restaurar Valores" para volver a la configuración inicial.
4. **Desactivar Script**: Presiona `F8` para desactivar el script.

=========================================
INFORMACIÓN ADICIONAL
=========================================
- El estado actual del script se muestra en la parte inferior de la interfaz o como un ToolTip temporal.
- Puedes personalizar los valores predeterminados y ajustar el script según tus necesidades.
